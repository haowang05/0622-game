# 素材语义索引（代码现状）

本索引用于统一“素材名 -> 语义 -> 触发条件/隐藏条件”。

## 1. 通用语义

- `背景.png`：场景主底图
- `亮背景.png / 原背景.png`：状态切换底图
- `X*.png`：弹窗关闭按钮
- `弹窗.png`：通用提示容器（如 S5 邮箱首次打开）
- `照片弹窗*.png`：照片容器
- `OS/os.png`：照片上的叙事叠层
- `*道具未选择.png`：背包默认态
- `*选择.png`：背包选中态
