# 游戏流程（代码对齐版）

本文件以当前代码为准（`code/src/main.js`，版本 `v0.3.32-20260701-0025`）。

## 1. 全局主线

`MainMenu -> Intro -> Office -> Dialog -> HouseExterior -> Mailbox -> HouseExterior -> LivingRoom(A/B/C循环) -> GardenEntry/GardenSky/PondFish -> Bedroom -> Password -> SofaRepair -> LivingRoomA(结局覆盖链路)`

> 说明：`EndingScene` 仍注册在引擎中，但当前主链路的结局实际在 `LivingRoomAScene` 内完成覆盖流程，不强制切换到 `EndingScene`。

## 2. 场景分段流程

### S0 `MainMenu`

- 显示背景与 3 个按钮（Start/Setting/Exit）。
- 进入时使用缓慢淡入（非黑色转场）。
- 背景 BGM：`前bgm(1)` 自 MainMenu 起播，至进入 `HouseExterior` 前持续。
- 左上角版本号支持键盘 `V` 显示/隐藏。
- `Start` 点击进入 `Intro`。
- `Setting/Exit` 当前为 stub（仅日志）。

### S1 `Intro`

- 使用 uncropped 资源叠加漫画帧：`1.png -> 2.png -> 3.png -> 4.png -> 5.png`。
- 进入时使用缓慢淡入；每叠加新帧时该帧单独淡入。
- 第 5 帧后显示 `open` 按钮。
- 点击 `open`：
  - `introCompleted=true`
  - 切换到 `Office`

### S2 `Office`

- 点击 `信.png`：
  - `letterRead=true`
  - 切到 `Dialog`

### S3 `Dialog`

- 点击 `accept 按钮.png`：
  - `commissionAccepted=true`
  - 切到 `HouseExterior`

### S4 `HouseExterior`

- 自本场景起切换 BGM 为 `后bgm2(1)`，直至游戏结束。
- 点击信箱进入 `Mailbox`。
- 点击门/抽屉类入口播放 `开门开抽屉声`。
- 大门逻辑：
  - 未解锁：需在背包选中 `key` 才能开门。
  - 开门后：`doorUnlocked=true`，消耗钥匙。
  - 门已开时点击进入 `LivingRoomA`。

### S5 `Mailbox`

- 首次点击邮箱：`邮箱关 -> 邮箱开`（`mailboxOpened=true`），弹出 `6/弹窗.png`（右上角 X 关闭）。
- 再点击获得钥匙：`inventory.key=true`。
- 回到 S4 后可选中钥匙开门。

## 3. 客厅循环（A/B/C）

### S6 `LivingRoomA`

- 书架点击：照片弹窗（暗态 `8/照片弹窗.png`，亮态 `22/照片弹窗.png`）。
- 沙发逻辑：
  - 未亮态：点击仅提示 `8/提示.png`。
  - 亮态且收齐 `wood+saw+hammer+nail`：
    - 第一次点击：显示 `22/修沙发/提示修沙发.png`
    - 第二次点击：进入 `SofaRepair`
  - 亮态且 `sofaRepaired=true`：不再进入修沙发。
- 修好后相机：
  - 点击 `22/相机.png` 打开 `22/修好沙发照片弹窗.png`
  - 关闭后：
    - `endingNotebookUnlocked=true`
    - 仍停留客厅，进入“笔记结局覆盖链路”
- 结局链路终点调整：保留到文本框提交阶段；点击 `文本框√.png` 后按钮盖章并结束，不再显示 `总结.png`。

### S7 `LivingRoomB`

- 客厅三面循环：
  - 暗态：`LivingRoomB ↔ LivingRoomA ↔ LivingRoomC`
  - 鱼塘/客厅点亮后：`LivingRoomA ↔ LivingRoomReturn`（`LivingRoomB` 从循环移除）
- 浇花成功播放 `浇水声`。
- 窗台浇花门禁：
  - 必须拥有并选中 `wateringCan`
  - 且 `waterFilled=true`
- 浇花成功后：
  - `flowersWatered=true`
  - 消耗水壶并取消水壶状态：
    - `inventory.wateringCan=false`
    - `wateringCanSelected=false`
    - `waterFilled=false`
  - 花园门立即可进入（无需切场景刷新）
- 相机拍照弹窗：`9,11/照片弹窗.png + 9,11/os.png`

### S8 `LivingRoomC`

- 橱柜流程：
  - `柜子 -> 开柜门壶在柜子里 -> 取壶 -> 空柜子`
  - 水壶获取后不自动选中。
- 接水流程：
  - 必须选中水壶
  - 必须点击 `10/开柜门壶在柜子里.png` 指定区域
  - 完成后 `waterFilled=true`
- 卧室门门禁（关键）：
  - 必须 `allGardenLit() && pondLitUp`
  - 否则点击无跳转
- 亮态替换：
  - 满足亮态条件后背景用 `17，21/背景.png`
  - 柜子/接水交互隐藏

## 4. 花园与池塘

### S9 `GardenEntry`

- 草坪点击进入 `GardenSky`。
- 池塘入口开启条件：`woodAcquired || allGardenLit`。
- 点亮态下保证池塘点击优先，不被草坪吞掉。
- 池塘在入口场景始终可见，仅状态切图（`池塘原/池塘亮`）。
- 返回规则：
  - 点亮前：返回 `LivingRoomB`
  - 鱼塘点亮后：返回 `LivingRoomA`（与“移除 LivingRoomB 循环”一致）

### S10 `GardenSky`

- 点亮谜题：花 5 + 云 3，任意顺序。
- 云朵在点亮前后均持续缓动，且位移幅度提升（更明显）。
- 全部点亮后背景切换 `13/亮背景.png`，并可出现相机。
- 木头拍照流程：
  - 相机弹窗含 `OS` 和木头图层。
  - 点击木头后：
    - `woodAcquired=true`
    - `inventory.wood=true`
    - 木头立即从当前弹窗消失
  - 弹窗仍需手动点 X 关闭（不自动关闭）。

### S12 `PondFish`

- 鱼逐条点亮（不是消失逻辑）。
- 全鱼点亮后出现锯子。
- 点击锯子后：
  - `inventory.saw=true`
  - `sawAcquired=true`
  - `pondLitUp=true`
  - 背景与池塘切到亮态（无缝替换）
- 返回规则：
  - 点亮前：返回按历史栈逻辑
  - 点亮后：返回箭头固定回 `LivingRoomA`

## 5. 卧室与密码

### S14 `Bedroom`

- 点击密码柜桌子进入 `Password`。
- 导航返回目标：
  - 暗态：`LivingRoomC`
  - 亮态：`LivingRoomReturn`

### S15 `Password`

- 锁定态背景：`19/背景.png + 19/密码0-9/空.png`。
- 默认密码 `0000`，点击四位循环数字。
- 输入 `2513` 后延迟约 2 秒，保持当前已输入画面；需再点击一次抽屉区域才真正开抽屉，并在该次点击播放开抽屉音效（不再回到 `0000`）。
- 开抽屉点击逻辑：
  - 第一次：日记弹窗
  - 第二次：一次性获取 `hammer` + `nail`，并 `passwordLooted=true`
- 领取后抽屉保持空态。

## 6. 修沙发流程（S16 `SofaRepair`）

- 弹窗主图：
  - 未修：`修沙发弹窗.png`
  - 已修：`修完沙发.png + 修好沙发os.png`
- 步骤状态机（严格顺序）：
  1. 选中木头并点击沙发 -> `sofaWoodPlaced`
  2. 选中锯子并点击屏幕 -> `sofaSawPlaced`
  3. 点击屏幕 -> `sofaWoodSplit`（显示木头分开）
  4. 点击屏幕 -> `sofaWoodResidual`（显示留下木料）
  5. 点击屏幕 -> `sofaWoodMounted`（显示吸附木料）
  6. 选中钉子点击屏幕 -> `sofaNailPlaced`
  7. 选中锤子点击屏幕 -> `sofaHammerPlaced`
  8. 点击屏幕 -> `sofaNailHits=1`（钉子2 + 锤子动）
  9. 点击屏幕 -> `sofaNailHits=2` + `sofaRepaired=true`
- 关闭修沙发弹窗后返回 `LivingRoomA`。
- 进入/关闭修沙发场景使用无黑幕快速切换；修沙发步骤推进不出现黑屏闪烁。

## 7. 结局覆盖链路（在 `LivingRoomA` 内）

触发前提：`endingNotebookUnlocked=true`（由“修好沙发照片弹窗关闭”设置）

流程：

1. 笔记按钮替换为 `打开的本子图标.png`
2. 点击后黑底覆盖并显示 `内容.png`
3. 点击屏幕 -> 第一枚 `盖章.png`
4. 点击屏幕 -> `跳转文本框提示.png`
5. 点击屏幕 -> 清空上层内容，显示 `文本框背景.png`
6. 点击屏幕 -> `文本框输入.png + 文本框√.png`
7. 键盘输入并显示闪烁光标
8. 点击 `文本框√.png`：
   - 保留输入文字
   - 按钮替换为 `文本框章.png`
   - 结束链路并置 `gameCompleted=true`
9. 不再显示 `总结.png`

## 9. 冻结规则（必须遵守）

1. 文档与实现冲突时，以代码状态机与条件判断为准。
2. S10 木头点击不得自动关闭弹窗。
3. 卧室门禁必须依赖 `allGardenLit && pondLitUp`。
4. S16 木料“残留”和“吸附”必须为两步点击。
5. 修好沙发照片关闭后不能黑屏，不能切离客厅。
6. 修沙发场景内步骤推进不得出现黑屏/过场 fade。
